Промпт

Ты — ведущий UI/UX дизайнер desktop-приложений уровня Apple, Linear, Arc Browser, Raycast, Notion и Figma.

Необходимо полностью переработать интерфейс существующей desktop-программы (Electron), не изменяя ни одной функции программы.

Главное требование

Запрещено менять бизнес-логику.

Нельзя:

менять структуру данных;
переименовывать переменные;
менять JavaScript;
менять обработчики событий;
менять генерацию документов;
менять работу Excel;
менять работу шаблонов.

Разрешается менять только внешний вид.

Стиль

Создать современный интерфейс в стиле Structural Minimalism + Bento Grid согласно трендам UI/UX 2026 года.

Интерфейс должен выглядеть как продукт уровня:

Linear
Raycast

Это не должен быть футуризм.

Это должен быть спокойный профессиональный интерфейс для ежедневной работы.
