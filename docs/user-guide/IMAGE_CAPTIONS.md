# Подписи к изображениям

1. **Запуск программы** (`images/01-launch.svg`) — раздел: 1. Общая информация. Старт приложения «Генератор договоров ГермесГарант» из Windows.
2. **Главное окно приложения** (`images/02-main-window.svg`) — раздел: 2. Интерфейс программы. Главное окно с левой областью данных и правой областью шаблонов.
3. **Обзор интерфейса с подписями** (`images/03-interface-callouts.svg`) — раздел: 2. Интерфейс программы. Основные зоны интерфейса: панель инструментов, формы, шаблоны, настройки и статус.
4. **Выбор шаблона** (`images/04-template-selection.svg`) — раздел: 4. Работа с шаблонами. Выбор одного или нескольких Word-шаблонов для генерации.
5. **Загрузка Excel-файла** (`images/05-excel-load.svg`) — раздел: 5. Работа с Excel. Диалог выбора и индикатор успешной загрузки Excel-файла.
6. **Заполнение основных данных** (`images/06-main-data.svg`) — раздел: 3. Порядок работы. Поля объекта недвижимости, продавца и покупателя.
7. **Работа с дополнительными полями** (`images/07-extra-fields.svg`) — раздел: 3. Порядок работы. Вкладки собственников и дополнительные данные сделки.
8. **Проверка введенной информации** (`images/08-data-check.svg`) — раздел: 3. Порядок работы. Кнопка «Проверить данные» и уведомление о результате проверки.
9. **Генерация документа** (`images/09-generation.svg`) — раздел: 6. Генерация документов. Запуск формирования выбранных документов.
10. **Сохранение документа** (`images/10-save-document.svg`) — раздел: 6. Генерация документов. Выбор папки сохранения и параметров именования файлов.
11. **Просмотр созданного документа** (`images/11-document-view.svg`) — раздел: 6. Генерация документов. Открытие готового DOCX-файла в Microsoft Word или другом редакторе.
12. **Возможные сообщения об ошибках** (`images/12-errors.svg`) — раздел: 7. Возможные ошибки. Примеры ошибок: не выбрана папка, не выбран шаблон, ошибка чтения Excel.
13. **Настройки приложения** (`images/13-settings.svg`) — раздел: 2. Интерфейс программы. Настройки генерации: папка, открытие после создания, добавление даты.
14. **Работа со справочниками** (`images/14-directories.svg`) — раздел: 11. Приложение. Справочники агентов, комиссий и плейсхолдеров в конфигурационных файлах.
15. **Работа с несколькими шаблонами** (`images/15-multiple-templates.svg`) — раздел: 4. Работа с шаблонами. Одновременный выбор нескольких документов.
16. **Полный пример работы** (`images/16-full-example.svg`) — раздел: 3. Порядок работы. Цепочка от загрузки Excel до получения готового документа.
17. **Общая схема работы приложения** (`images/17-workflow.svg`) — раздел: 11. Приложение. Excel → форма → выбор шаблонов → DOCX.
18. **Блок-схема действий пользователя** (`images/18-flowchart.svg`) — раздел: 3. Порядок работы. Последовательность решений и действий пользователя.
19. **Схема движения данных** (`images/19-data-flow.svg`) — раздел: 5. Работа с Excel. Данные Excel преобразуются в плейсхолдеры Word.
20. **Инфографика «Быстрый старт»** (`images/20-quick-start.svg`) — раздел: 9. Советы по работе. Памятка по минимальному набору действий.
21. **Памятка «Типичные ошибки»** (`images/21-common-errors.svg`) — раздел: 7. Возможные ошибки. Частые причины сбоев и быстрые решения.
22. **Памятка «Советы по эффективной работе»** (`images/22-tips.svg`) — раздел: 9. Советы по работе. Рекомендации по подготовке данных и шаблонов.
